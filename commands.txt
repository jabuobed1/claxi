
1. BEFORE SWITCHING TO CODEX (VERY IMPORTANT)

---

Run:

git status
git add .
git commit -m "Commit before pull - 4."
git push origin main

(This ensures Codex sees your latest work from GitHub)

2. AFTER CODEX HAS MADE CHANGES (BACK TO CODESPACES)

---

Run:

git status
git fetch origin
git pull origin main
